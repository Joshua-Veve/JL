# Digital Library Catalogue System

## Project Overview
This is a web-based digital library catalogue system for Academic City University. It allows students to register, login, search for books, borrow and return books. Admins can manage the book inventory.

## Features
- User registration and authentication
- Book catalogue with search and filter
- Borrow and return books
- Admin panel for book management

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express, PostgreSQL
- Authentication: JWT
- Deployment: Backend on Render, Frontend on GitHub Pages
